# Visão geral

Depois de entrar, a barra lateral dá acesso a todas as áreas do sistema:

| Ícone | Área | O que faz |
|---|---|---|
| ☀️ | **Meu Dia** | Painel do dia: progresso, tempo planejado/restante, tarefas de hoje. |
| ✅ | **Tarefas** | Lista completa com filtros por status e busca. |
| 📅 | **Agenda** | Calendário dia/semana/mês com arrastar-e-soltar. |
| 📋 | **Kanban** | Quadro visual por coluna. |
| ⏱️ | **Pomodoro** | Cronômetro de foco. |
| 📁 | **Projetos** | Iniciativas com prazo e % concluído. |
| 🎯 | **Metas** | Objetivos vinculados a tarefas. |
| 🏷️ | **Categorias** | Classificação temática das tarefas. |
| 👤 | **Papéis** | Trabalho, Estudos, Família... |
| ⚙️ | **Configurações** | Tema, idioma, formato de hora, padrões de Pomodoro. |

A barra lateral pode ser recolhida (ícone de seta, no topo) para ganhar
espaço — a preferência fica salva no navegador. No celular, ela vira um menu
deslizante acessível pelo ícone de hambúrguer no topo.

O tema (claro/escuro/automático) pode ser alternado a qualquer momento pelo
ícone de meia-lua no topo, ou fixado em **Configurações**.
