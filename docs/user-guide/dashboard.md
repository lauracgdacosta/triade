# Meu Dia

A tela inicial (`/dashboard`) mostra:

- **Saudação** contextual (Bom dia/Boa tarde/Boa noite) e a data de hoje.
- **Cartões de indicadores**: tarefas de hoje, quantas já foram concluídas,
  tempo planejado e tempo restante (planejado menos já realizado).
- **Barra de progresso** do dia (% de tarefas concluídas).
- **Lista das tarefas de hoje**, com prioridade, horário e duração.

O tempo restante é recalculado a cada tarefa marcada como concluída ou
reaberta, e considera apenas as tarefas com data de hoje.
