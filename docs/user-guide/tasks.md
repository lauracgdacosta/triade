# Tarefas

Em **/tasks**, as abas no topo filtram por status: **Pendente**, **Em
andamento**, **Aguardando**, **Concluído**, **Cancelado**, **Arquivado**. A
busca na barra superior filtra por título/descrição/observações.

## Criar ou editar

Clique em **+ Nova tarefa** (ou no título de uma tarefa existente) para abrir
o formulário, com:

- **Título** (obrigatório) e **descrição rica** (negrito, itálico, listas).
- **Observações** (texto livre adicional).
- **Data**, **horário** e **duração prevista**.
- **Prioridade**: Importante, Urgente, Circunstancial ou Sem prioridade.
- **Categoria**, **Projeto**, **Papel** e **Meta** (todos opcionais).
- **Cor**, **localização** e **etiquetas** (múltiplas).

## Ações rápidas

No menu de três pontos de cada tarefa:

| Ação | Efeito |
|---|---|
| **Editar** | Reabre o formulário preenchido. |
| **Duplicar** | Cria uma cópia (`"(cópia)"` no título), mesmas etiquetas. |
| **Cancelar** | Marca como cancelada (não some da lista, some das abas ativas). |
| **Arquivar** | Marca como arquivada. |
| **Excluir** | Remove definitivamente (e seus anexos). |

Marcar o círculo à esquerda do título conclui/reabre a tarefa diretamente,
sem abrir o formulário.

## Anexos

Anexos são enviados para o Supabase Storage e ficam vinculados à tarefa;
removê-los também remove o arquivo do Storage.
