# Agenda

Em **/agenda**, alterne entre as visões **Mês**, **Semana** e **Dia** pelos
botões no canto superior direito do calendário.

- **Criar um compromisso**: clique (ou arraste) em um horário vazio, ou use o
  botão **+ Novo compromisso**.
- **Mover**: arraste o compromisso para outro horário/dia.
- **Redimensionar**: arraste a borda inferior para ajustar a duração.
- **Editar**: clique no compromisso para reabrir o formulário.
- **Recorrência**: preencha o campo RRULE (ex.: `FREQ=WEEKLY;BYDAY=MO,WE,FR`
  para segundas, quartas e sextas) para repetir o compromisso.
- **Conflito de horário**: se o novo horário sobrepõe outro compromisso, um
  aviso aparece — o compromisso é salvo mesmo assim (a decisão de resolver o
  conflito é sua).

Cada compromisso pode ser vinculado a uma **categoria** e/ou **projeto**, e
tem cor própria para facilitar a identificação visual no calendário.
